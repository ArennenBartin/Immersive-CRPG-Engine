# THE THIRD VOICE — MASTER BUILD PLAN v1
### The concrete, numbered, agent-facing build document. One place to work from.

**Sources of truth:** `The_Third_Voice_GDD_v0_5.md` (design), `the_third_voice_stage_play_v2_2.md` (dialogue + staging, quoted verbatim below), `AI_GAME_BUILD_GUIDE_FOR_CODEX` (engine contract). Where this plan and those docs conflict, this plan wins for build order; the play wins for dialogue wording; the engine guide wins for schema/field names.
**Target:** ≥45 minutes of real main-story gameplay (budget in Phase 9). Jam deadline July 26.
**Prime directive (from the engine guide):** express everything in package data — maps, dialogue, switches, conditions, cutscenes, triggers, quests, shops, barks, regions — before touching engine code. This plan requires exactly TWO small engine edits (steps 22, 24); everything else is authoring.

**Global implementation decisions (read before step 1):**
- **UNDERSTANDING** is a hidden numeric score implemented as **faction reputation** on a hidden faction `the_road`. Raise/lower it with `adjust_faction_rep` in cutscenes. Gate content with `faction: the_road, rep_gte / rep_lte` conditions. Never show it in UI.
- **ATTEND** is a **dialogue subtree pattern**, not a new system: an "Attend…" option on a target's dialogue root, gated `not attended_<target>`, which immediately sets `attended_<target> = true` on selection (one shot, forever). Inside: authored options secretly typed TRUE / GRID / SURFACE / EXIT. Outcomes: **Success** (+rep, unlocks deeper lines/flags), **End Early** (neutral), **Fail** (regular = neutral; spectacular = −rep). This is the mechanic, exactly: a dialogue you can succeed, end early, or fail, once per target.
- **ATTENDABLE SHADOWS** are staged as **NPC-first encounters**: placed as non-hostile entities whose interact opens their Attend; a spectacular fail or the player attacking triggers `start_combat` via cutscene; Success flips them permanently peaceful (switch) and they become talkable. Mandatory enemies are plain hostiles. True mid-combat Attend is a stretch slice (step 25), not on the critical path.
- **SUPERTITLES / MISSION CARDS** are blocking cutscenes: `play_sound` (sting) → `show_dialogue` node whose speaker is `✦` and whose scene image is a generated title card.
- **PAUSES** are cutscene `wait` actions, specified in milliseconds at every beat below. When the play says *(beat)*, use `wait: 900` unless a step says otherwise.
- **Switch naming:** durable facts, lowercase snake: `voice_heard`, `attended_claro7`, `nell_shadows_spared`, `jory_met`, `room7_entered`, `mender_severed`.

## PHASE 1 — MECHANICS SLICES (the only new "systems," mostly data)
